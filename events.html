LG's Remastered Collection & Plus Bundle / Watch the new trailer!!

INFO ABOUT THE EVENT

LG's Remastered Collection will contain three small remasters in one; Candy Bar Clicker, -Ball-, and Two-Player Ball Wars. These projects were some of my first, back when I joined Scratch in 2020.

The Plus Bundle is two separate projects bound together, an overhaul update to Infinity Player+, and the release of EggWars+, which is actually condensed into one sprite!

I've been meaning to release all of these projects for a while now, and now I feel is a great time to do so, especially since I want to get them out of the way and work on George 1.5 and SSMB.

This complete package will release on February 25th, 2024.
